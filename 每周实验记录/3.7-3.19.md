张起给的变量代换的demo：
```python
import numpy as np

if __name__ == '__main__':
    n = 10
    u = np.zeros(n)

    # forward
    # u -> w
    w = np.exp(u)
    sw = w.sum()
    w /= sw

    # backward
    # u -> w -> dw / du
    grad = np.diag(w) - np.exp(u.reshape((1, n)) + u.reshape((n, 1))) / sw ** 2

    print('u =', u)
    print('w =', w)
    print('dw / du =', grad)
```
